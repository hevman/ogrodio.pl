# content/

Treści artykułów w formacie JSON.

## Struktura folderów

### `articles/`
Porady ogrodnicze — uprawa roślin, choroby, szkodniki, pielęgnacja.
Slug URL: `/porady/[slug]`
Przykład: `borowka-choroby-lisci-diagnoza.json`

### `articles/calendar/`
Seria "Ogród w [miesiącu]" — co robić w ogrodzie każdego miesiąca.
Slug URL: `/ogrod-w/[miesiac]`
Pliki: `ogrod-w-styczniu.json`, `ogrod-w-lutym.json`, ..., `ogrod-w-grudniu.json`

### `articles/sowing/`
Seria "Co siejemy i sadzimy w [miesiącu]" — kalendarz siewu i sadzenia.
Slug URL: `/siejemy-sadzimy/[miesiac]`
Pliki: `co-siejemy-w-styczniu.json`, `co-siejemy-w-lutym.json`, ..., `co-siejemy-w-grudniu.json`

### `articles/guides/`
Przewodniki tematyczne — Trawnik, Warzywnik, Projektowanie ogrodów, itd.
Slug URL: `/przewodniki/[slug]`
Przykład: `trawnik-od-podstaw.json`, `warzywnik-dla-poczatkujacych.json`
